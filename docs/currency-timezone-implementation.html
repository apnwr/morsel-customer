3. LocaleContext Internal Flow

    src/contexts/LocaleContext.tsx
    ┌────────────────────────────────────────────────────┐
    │                                                    │
    │  state                                              │
    │  ├── currency  = useState("MUR")                   │
    │  └── timezone  = useState("Indian/Mauritius")       │
    │                                                    │
    │  setLocale({ currency?, timezone? })               │
    │  ├── no deps  → stable ref, never triggers re-renders│
    │  └── uses functional setState to skip no-op updates │
    │                                                    │
    │  context value  =  useMemo([currency, timezone])    │
    │  ├── currency                                        │
    │  ├── timezone                                        │
    │  ├── formatPrice(amount)  → "Rs 250.00"             │
    │  └── setLocale                                       │
    │                                                    │
    └────────────────────────────────────────────────────┘

        Component usage:

        const { formatPrice } = useLocale();
        formatPrice(250)    →  "Rs 250.00"
        formatPrice(12.5)   →  "Rs 12.50"

5. Price Format

    formatPrice(amount, currencyCode?)  →  "{symbol} {amount.toFixed(2)}"

    formatPrice(250,   "MUR")  →  "Rs 250.00"
    formatPrice(12.5,  "USD")  →  "$ 12.50"
    formatPrice(9.99,  "EUR")  →  "€ 9.99"
    formatPrice(500,   "INR")  →  "₹ 500.00"
    formatPrice(100)           →  "Rs 100.00"   ← MUR default

    Addon modifier format:
    +{formatPrice(50)}         →  "+Rs 50.00"

7. Defaults

Field Default value Reason
currency MUR — Rs Mauritius market default
timezone Indian/Mauritius (UTC+4) Mauritius market default
Fallback rule: If the session API does not return currency or timezone, the app silently stays on the Mauritius defaults. No loading state, no flash — defaults are applied synchronously before the first render.

8. How to Use in a New Component

    // 1. Import the hook
    import { useLocale } from '@/contexts/LocaleContext';

    // 2. Destructure inside your component
    const { formatPrice } = useLocale();

    // 3. Use everywhere a price is displayed
  <span>{formatPrice(item.price)}</span>          // "Rs 250.00"
  <span>{formatPrice(cart.total)}</span>          // "Rs 1,200.00"

    // 4. For timezone — display a UTC timestamp in branch local time
    const { timezone } = useLocale();
    const formatted = new Intl.DateTimeFormat('en-US', {
    timeZone: timezone,
    hour: 'numeric',
    minute: '2-digit',
    hour12: true,
  }).format(new Date(utcTimestamp));
